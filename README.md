Proyecto Autora – Plataforma de Aprendizaje Infantil

Aplicación educativa interactiva diseñada para ayudar a niños a reconocer letras, mejorar la memoria y construir palabras de forma progresiva y adaptativa.

Este proyecto no busca solo entretener, sino reforzar habilidades reales de aprendizaje, especialmente en niños que necesitan más repetición, acompañamiento y estímulo positivo.

🧠 Objetivo

Desarrollar un sistema de juegos conectados que:

Refuerce el reconocimiento de letras
Mejore la velocidad de respuesta
Desarrolle la construcción de palabras
Mantenga motivación mediante recompensas significativas
Se adapte al progreso del niño
🎮 Juegos Incluidos
1. Aventura del Alfabeto
Selección de letras
Refuerzo visual y auditivo
Base del aprendizaje
2. Letras Descendiendo
Letras caen desde la pantalla
El jugador debe seleccionar la correcta
Incluye:
❤️ Corazones (vidas extra)
💣 Bombas (penalización)
Aumenta dificultad con niveles
3. Palabras Ocultas
Construcción de palabras a partir de letras mezcladas
Niveles progresivos según dificultad
Refuerza memoria y secuencia
🌐 Idiomas

Soporte bilingüe:

Español 🇪🇸
Inglés 🇺🇸

El idioma se aplica de forma global en todos los juegos.

🏆 Sistema de Recompensas
No basado en puntos fáciles
Recompensas por logros reales
Ejemplos:
Completar niveles sin errores
Formar palabras correctamente
Mantener rachas positivas

Coleccionables:

🦄 Unicornio
🦥 Oso perezoso
🐹 Capibara
🧩 Arquitectura

El sistema utiliza un núcleo central:

gameCore
Maneja estado global
Controla vidas, puntaje y progreso
Unifica los juegos
i18n
Manejo de idiomas
Persistencia:
localStorage para guardar progreso
💡 Filosofía del Proyecto

Este proyecto nace desde una necesidad real:

Ayudar a un niño que está teniendo dificultades para reconocer y escribir letras.

Por eso el diseño sigue principios clave:

Sin castigos agresivos
Repetición inteligente
Refuerzo positivo
Progresión gradual
